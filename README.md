# Signature PDF Generator

A Node.js & TypeScript project to automate filling PDF forms with dynamic order, customer, vehicle, and finance data. It also supports adding signature tags for electronic signing.

## Features
- Fill PDF forms automatically
- Add live current date
- Insert signature placeholders
- Works with multiple pages

## Usage
Run any TypeScript file dynamically with:

```bash
npm run dev -- src/orderForm.ts
npm run dev -- src/addSignatureTag.ts
